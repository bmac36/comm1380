Presenting Information and Social Media

Activity: Putting It All Together

We’ve covered how to structure and present information using headings, we’ve learned the difference between functional and descriptive headings, and we know the difference between a conclusion and a recommendation. This task is designed to pull all those pieces together.

Scenario

You have been asked to pick a restaurant for a year-end dinner celebration for a health care centre and need to organize the information you’ve collected and present it to your coworkers. Your job in this task is to structure it using proper headings so that readers understand the purpose of the report, why you’re writing it, the results of the research, and your conclusion.

Expand to see the Instructions

Instructions

  1. Read the text in the box below. It contains no chunking of information or headings
  2. Assume the content is in the correct order
  3. Break up this report into digestible chunks
    • Use functional headings at the beginning and end of the report
    • Use Descriptive headings for the body of the report
  4. Place all headings flush left and indicate whether it is a heading or subheading by using the following font characteristics
    • 1st Level: Header 1
    • 2nd Level: Header 2
    • 3rd Level: Header 3
  5. Choose a heading from the following list and add it to the report as either a heading or subheading in the appropriate location. You can use some headings more than once.
    • Background
    • Conclusions
    • Introduction
    • Option 1: Copper Kettle Restaurant
    • Option 2: Michi Japanese Restaurant and Sushi Bar
    • Option 3: Memories Fine Dining
    • Location
    • Methods
    • Menu
    • Prices
    • Purpose
    • Ratings and Review
    • Restaurant Options
    • Rooms Available
  6. Once you have finished editing the text to include headings, click Save.
